# Privacy Policies

Static privacy-policy pages for applications published by Mystery Box.

## Current policies

- Deduce It: `deduce-it/index.html`

## GitHub Pages setup

In this repository, open **Settings → Pages** and choose:

- Source: **Deploy from a branch**
- Branch: **main**
- Folder: **/(root)**

After deployment, the site URLs will be:

- Policy index: `https://mastermystery007.github.io/privacy_policies/`
- Deduce It: `https://mastermystery007.github.io/privacy_policies/deduce-it/`

## Adding another app

1. Create a new folder using a lowercase URL-friendly app name.
2. Add an `index.html` privacy policy inside that folder.
3. Add the app to the root `index.html` list.
4. Keep each policy accurate for the SDKs and data practices used by that app.
